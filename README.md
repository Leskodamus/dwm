# Fork of dwm

The [suckless dwm (dynamic window manager)](https://dwm.suckless.org/).

## Patches applied

+ actualfullscreen
+ autostart
+ cfacts \+ vanitygaps
+ hide vacant tags
+ noborderfloatingfix
+ pertag
+ resetlayout
+ resizecorners
+ scratchpad
+ stacker 
+ statusallmons
+ systray

## Configuration

Simply modify the `config.h` file. 
Default configuration is stored in `config.def.h`.

## Installation

```
sudo make clean install
```
